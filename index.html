I like programming

Here's a few of my favourite languages.

Python

Python's my all-time favourite language, and it's where I take my name from. It's simple and easy to learn, requires minimal code, but still comes packed to the brim with tons of features and libraries. Python isn't the best language for GUI applications due to its lack of native GUI support, but it's still a great language.

C#

I've been learning this language for a while, and it's pretty great. It's going to get more and more useful as Microsoft's Universal App platform -- which lets you create apps that run on the entire Microsoft platform -- develops. Its syntax is extremely conventional, making it easy to learn for experienced programmers. Overall, it's a pretty great language, and its integration into the .NET framework makes it easy to create and deploy large-scale applications.

Scratch

Though Scratch is mainly a language designed for beginners and kids learning to program, I still program in and enjoy Scratch a lot. Why? Because Scratch's easy- and quick-to-use interface, with integrated 2D graphics, makes it simple to rapidly make and complete new projects. In addition, Scratch's huge online community is a huge plus; you can share your projects on the web and instantly receive feedback from your peers.
